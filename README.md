# CV-Analyzer (Monorepo)

Projeto profissional de análise de currículos com IA.

## Estrutura

- `/backend`: API em Django
- `/frontend`: Interface em React

## Setup Rápido

```bash
# Backend
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Frontend
cd frontend
npm install
npm run dev
